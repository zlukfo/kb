Как применять

Запросы API могут выполняться с помощью различных языков программирования. Здесь рассматривается Python и его библиотека fast_bitrix24

Полезные ссылки

  1. Битрикс24 API
  2. Python-библиотека fast_bitrix24
  3. Структура таблиц сущностей

Для выполнения API-запрсов скрипт должен инициализировать подключение к порталу

from conf import WEBHOOK
from fast_bitrix24 import Bitrix

def main(btx):
    # pass

if __name__ == '__main__':
    btx = Bitrix(WEBHOOK, respect_velocity_policy=True)
    main(btx)

где WEBHOOK - адрес входящего вебхука, предварительно созданного на целевом портале
