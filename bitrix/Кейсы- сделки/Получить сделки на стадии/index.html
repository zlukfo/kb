Получить сделки на стадии

Для чего может понадобиться

  • перенести сделки на другую стадию
  • удалить сделки со стадии
  • изменить значения полей сделок на стадии
def task4(btx):
    result = btx.get_all(
        'crm.deal.list', 
        {
            'filter': {"STAGE_ID": ["3", "7"] },
            'select': [ "ID"]
        }
    )
    print (len(result))

где STAGE_ID - id стадии, одной или несколько (как узнать - см. 1.2)

В select указывается набор id полей которые нужно возвращать. По возможности указывать только нужные поля, чем менньше полей, тем быстрее отрабатывает запрос.
