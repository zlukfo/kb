Лиды: статусы

all_lead_status = btx.get_all('crm.status.list',{'filter':{'ENTITY_ID':'STATUS'}})

Возвращает список элементов-словарей с информацией о статусе. Наиболее востребованные ключи - ['ID'] - уникальный идентификатор статуса - ['STATUS_ID'] - id статуса, указывается в лиде определяя его принадлежность к статусу - ['NAME'] - название статуса - ['EXTRA']['SEMANTICS'] - категория статуса "process" - все рабочие статусы "success" - успешный статус "failure" - системный провальный статус "apology" - пользовательский провальный статус

Лиды связаны со статусом через STATUS_ID. Именно этот параметр передается в фильтр для отбора лидов. Поэтому к наиболее частым кейсам онтосятся

Получить список всех STATUS_ID из заданных категорий

target_category = ["apology", "failure"]
target_status_id = [i['STATUS_ID'] for i in status if i['EXTRA']['SEMANTICS'] in target_category]

Получить STATUS_ID по названию статуса

target_name = ["недозвон", "Провал 2"]
target_status_id = [i['STATUS_ID'] for i in status if i['NAME'] in target_name]
