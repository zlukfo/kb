Лиды: доп поле список

LIST_FIELD_ID = 'id'
field_list = a.get_all('crm.lead.fields')[LIST_FIELD_ID]['items']

У каждого значения списка есть его ID и строковое значение. Когда в диле мы хотим присвоить полю значение из списка - в него нужно записать ID. Поэтому зачастую полученный список лучше хранить так

field_list = {i['VALUE']: i['ID'] for i in field_list}

и искать ID элемента по ключу словаря
